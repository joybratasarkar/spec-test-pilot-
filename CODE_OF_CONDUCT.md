# Code of Conduct

## Our Standards

We are committed to a respectful, inclusive, and constructive community.

Examples of behavior that contributes to a positive environment:

1. Respectful technical discussion
2. Constructive feedback
3. Openness to different viewpoints and experience levels
4. Assuming good intent while reviewing code

Examples of unacceptable behavior:

1. Harassment or personal attacks
2. Discriminatory or derogatory language
3. Deliberate intimidation
4. Bad-faith disruption of community collaboration

## Scope

This Code of Conduct applies to project spaces including issues, pull requests, and discussions.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, and contributions that violate this Code of Conduct.

## Reporting

Report concerns privately to project maintainers via repository contact channels.
